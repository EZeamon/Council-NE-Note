🟠 HubSpot Integration

Push leads directly into HubSpot as contacts. Requires a Private App token + a free Cloudflare Worker proxy (fixes browser CORS restrictions).

Step 1 — HubSpot Private App Token
1
In HubSpot → Settings → Integrations → Private Apps → Create
2
Name it "EZ Smart CRM", enable scopes: crm.objects.contacts.read + write
3
Click Create app → copy the token starting with pat-na2-...
Step 2 — Cloudflare Worker Proxy URL
1
Go to cloudflare.com → sign up free → Workers & Pages → Create Worker
2
Name it ez-smart-hs-proxy → Deploy → Edit Code → paste the Worker code (download below) → Save & Deploy
3
Copy your worker URL — looks like ez-smart-hs-proxy.yourname.workers.dev
